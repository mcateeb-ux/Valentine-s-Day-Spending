# Valentine-s-Day-Spending
R HW#2 Introduction to R Tidyverse in DataCamp (Quiz).
Completed for ADAN|ADEC7900.

The script analyzes and plots survey data on how consumers celebrate Valentine's Day compiled by the National Retail Federation in the United States.

Data source: https://www.kaggle.com/datasets/joebeachcapital/valentines-day-consumer-data.
